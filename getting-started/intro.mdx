The Baseline API is a modern, RESTful interface built to help lending platforms, CRMs, underwriting tools, and custom workflows interact programmatically with the Baseline private lending system. Whether you're creating loans, syncing borrower data, managing documents, or integrating external systems into your lending lifecycle, the Baseline API gives developers direct, secure access to the same powerful infrastructure that drives Baseline's core product.

Baseline itself is a comprehensive software platform for private lenders that streamlines the loan lifecycle from origination through servicing, investor engagement, and borrower experience — all in one unified system. This holistic view means the API isn't just a data storefront; it's the connective tissue for automating, integrating, and extending your lending operations.

## Base URL

The base URL for all API requests is:

```text
https://production.baselinesoftware.com/production/api
```
