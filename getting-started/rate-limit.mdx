---
title: "Rate Limits"
description: "Understanding API rate limits"
---

API requests are rate limited to **700 requests per 5 minutes** per API key. This limit can be adjusted at any time.

If you exceed this limit, subsequent requests will be blocked until the window resets.

## Response

When rate limited, the API will return:

```text
403 Forbidden
```

## Best Practices

- Batch requests where possible instead of making many individual calls.
- Cache responses that don't change frequently.
- Use the [Graph API](/everything-api/graph-api) to retrieve multiple related records in a single request instead of making separate calls for each record.

<Tip>
  If you need higher rate limits for your use case, contact the Baseline team.
</Tip>
