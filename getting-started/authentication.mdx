---
title: "Authentication"
description: "How to authenticate with the Baseline API"
---

All API requests must include an API key in the `Authorization` header.

## Header Format

```text
Authorization: Token <your-api-key>
```

## Example Request

```bash
curl -X GET https://production.baselinesoftware.com/production/api/loan \
  -H "Authorization: Token your-api-key-here"
```

## Base URL

```text
https://production.baselinesoftware.com/production/api
```

## Error Responses

If your API key is missing, invalid, or inactive, you will receive a `403` response:

```text
403 Not authorized
```

## Getting an API Key

API keys are provisioned by the Baseline team. See [Requesting API Keys](/getting-started/requesting-api-keys) for details.

<Warning>
  Keep your API key secure. Do not expose it in client-side code or public repositories.
</Warning>
