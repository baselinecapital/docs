---
title: "Requesting API Keys"
description: "How to get access to the Baseline API"
---

To get access to the Baseline API, please reach out to the Baseline team directly. API keys are provisioned on a per-client basis.

## Contact

Email the Baseline team at [**support@baselinesoftware.com**](mailto:support@baselinesoftware.com) to request API access. Please include:

- Your company name
- A description of your integration use case
- The environment you need access to (production, staging)

Once approved, you will receive an API key that you can use to authenticate your requests.
