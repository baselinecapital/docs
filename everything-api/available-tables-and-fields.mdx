A complete reference of all tables and fields available through the Graph API. You can also discover these dynamically via [introspection](/everything-api/graph-api#introspection).

## Core Lending

**loans** `id`, `number`, `name`, `amount`, `status`, `substatus`, `date_closing`, `use_of_funds`, `program`, `frequency`, `regular_payment`, `date_firstpayment`, `date_maturity`, `date_paidoff`, `date_application_submitted`, `date_application_requested`, `term`, `grace_days`, `grace_method`, `due_day`, `rate`, `loan_type`, `date_denied`, `date_status_change`, `date_substatus_change`, `product_id`, `application_id`, `signature_id`, `created_at`, `updated_at`

**payments** `id`, `date`, `amount`, `type`, `method`, `notes`, `date_release`, `date_reversed`, `date_paidto`, `date_due`, `loan_id`, `person_id`, `admin_id`, `created_at`, `updated_at`

**transactions** `id`, `date`, `debit`, `credit`, `amount`, `gl`, `is_reversed`, `is_paidout`, `date_paidout`, `order`, `loan_id`, `payment_id`, `funding_source_id`, `person_id`, `charge_id`, `created_at`, `updated_at`

**charges** `id`, `name`, `amount`, `date_charged`, `date_due`, `loan_id`, `person_id`, `field_id`, `created_at`, `updated_at`

**schedules** `id`, `due`, `period`, `principal`, `interest`, `to_trust`, `balance`, `is_next`, `is_payoff`, `loan_id`, `created_at`, `updated_at`

**funding\_sources** `id`, `class`, `rate`, `original_amount`, `committed_amount`, `date_effective`, `date_funded`, `date_firstinvested`, `loan_id`, `investor_id`, `offering_id`, `created_at`, `updated_at`
