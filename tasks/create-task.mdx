---
title: "Create a new task"
description: "Create a new task in Baseline"
api: "POST /task"
---


## Request Body

<ParamField body="Name" type="string">
  Task name
</ParamField>

<ParamField body="Description" type="string">
  Task description
</ParamField>

<ParamField body="Date_Due" type="string">
  Due date
</ParamField>

<ParamField body="Status" type="enum">
  `To Do`, `In Progress`, `Done`, `Not Required`
</ParamField>

<ParamField body="Loan_Id" type="string">
  ID of a loan to associate with
</ParamField>

<ParamField body="Subtasks" type="array">
  Array of subtask objects
</ParamField>

## Example Response

```json
{
  "task": {
    "Id": "1234",
    "Name": "Validate credit score",
    "Description": "Check borrower's credit score validity",
    "Status": "To Do",
    "Date_Due": "2024-12-12",
    "Loan_Id": "5678",
    "Subtasks": [{ "Done": false, "Name": "Archive a credit score report" }]
  }
}
```
