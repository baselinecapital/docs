---
title: "Get a task"
description: "Retrieve a single task by ID"
---

`GET /task/{id}`

## Path Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `id` | string | Task ID |

## Example Response

```json
{
  "task": {
    "Id": "1234",
    "Name": "Validate credit score",
    "Description": "Check borrower's credit score validity",
    "Status": "To Do",
    "Date_Due": "2024-12-12",
    "Loan_Id": "5678",
    "Subtasks": [{ "Done": false, "Name": "Archive a credit score report" }]
  }
}
```
