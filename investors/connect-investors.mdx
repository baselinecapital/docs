---
title: "Connect an investor to another investor"
description: "Will connect a person to a company or vice versa"
api: "PUT /investor/{id}/connect/{to}"
---


## Path Parameters

<ParamField path="id" type="string">
  ID of an investor
</ParamField>

<ParamField path="to" type="string">
  ID of an investor to connect to
</ParamField>

## Example Response

```json
{
  "investor": {
    "Id": "1234",
    "Name": "John Smith",
    "Companies": [
      { "Id": "4567", "Name": "Acme Co" }
    ]
  }
}
```
