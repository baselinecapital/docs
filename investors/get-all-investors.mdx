---
title: "Get all investors"
description: "Retrieve a paginated list of all investors"
---

`GET /investor`

## Query Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `page` | object | Pagination parameters |

## Example Response

```json
{
  "investors": [{
    "Id": "1234",
    "Name": "John Smith",
    "Email": "john.smith@mailsac.com",
    "Is_Company": false
  }],
  "pagination": { "total": 1, "page_size": 10, "page": 0 }
}
```
