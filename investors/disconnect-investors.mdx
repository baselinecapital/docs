---
title: "Delete an investor from another investor"
description: "Will disconnect a person from a company or vice versa"
---

`DELETE /investor/{id}/connect/{to}`

## Path Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `id` | string | ID of an investor |
| `to` | string | ID of an investor to disconnect from |

## Example Response

```json
{
  "investor": {
    "Id": "1234",
    "Name": "John Smith",
    "Companies": []
  }
}
```
