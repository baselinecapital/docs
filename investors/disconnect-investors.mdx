---
title: "Delete an investor from another investor"
description: "Will disconnect a person from a company or vice versa"
api: "DELETE /investor/{id}/connect/{to}"
---


## Path Parameters

<ParamField path="id" type="string">
  ID of an investor
</ParamField>

<ParamField path="to" type="string">
  ID of an investor to disconnect from
</ParamField>

## Example Response

```json
{
  "investor": {
    "Id": "1234",
    "Name": "John Smith",
    "Companies": []
  }
}
```
