---
title: "Get an investor"
description: "Retrieve a single investor by ID"
api: "GET /investor/{id}"
---


## Path Parameters

<ParamField path="id" type="string">
  ID of an investor
</ParamField>

## Example Response

```json
{
  "investor": {
    "Id": "1234",
    "Name": "John Smith",
    "Email": "john.smith@mailsac.com",
    "Is_Company": false
  }
}
```
