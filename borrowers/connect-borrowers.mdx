---
title: "Connect a borrower to another borrower"
description: "Will connect a person to a company or vice versa"
api: "PUT /borrower/{id}/connect/{to}"
---


## Path Parameters

<ParamField path="id" type="string">
  ID of a borrower
</ParamField>

<ParamField path="to" type="string">
  ID of a borrower to connect to
</ParamField>

## Example Response

```json
{
  "borrower": {
    "Id": "1234",
    "Name": "John Smith",
    "Companies": [
      {
        "Id": "4567",
        "Name": "Acme Co"
      }
    ]
  }
}
```
