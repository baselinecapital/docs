---
title: "Delete a borrower from another borrower"
description: "Will disconnect a person from a company or vice versa"
api: "DELETE /borrower/{id}/connect/{to}"
---


## Path Parameters

<ParamField path="id" type="string">
  ID of a borrower
</ParamField>

<ParamField path="to" type="string">
  ID of a borrower to disconnect from
</ParamField>

## Example Response

```json
{
  "borrower": {
    "Id": "1234",
    "Name": "John Smith",
    "Companies": []
  }
}
```
