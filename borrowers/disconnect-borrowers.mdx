---
title: "Delete a borrower from another borrower"
description: "Will disconnect a person from a company or vice versa"
---

`DELETE /borrower/{id}/connect/{to}`

## Path Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `id` | string | ID of a borrower |
| `to` | string | ID of a borrower to disconnect from |

## Example Response

```json
{
  "borrower": {
    "Id": "1234",
    "Name": "John Smith",
    "Companies": []
  }
}
```
