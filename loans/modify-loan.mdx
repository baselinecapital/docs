---
title: "Modify an existing loan"
description: "Update fields on an existing loan"
api: "PATCH /loan/{id}"
---


All fields are optional. Any field available in the loan can be set via a body parameter using its name. For example, a field named "Foo Bar" can be set by sending `{Foo_Bar:1234}` in the request.

## Path Parameters

<ParamField path="id" type="string">
  Loan ID
</ParamField>

## Request Body

<ParamField body="Name" type="string">
  Loan name
</ParamField>

<ParamField body="Status" type="enum">
  `lead`, `processing`, `underwriting`, `approved`, `closed`, `servicing`, `archived`
</ParamField>

<ParamField body="Borrower_Name" type="string">
  Borrower name
</ParamField>

<ParamField body="Borrower_Email" type="string">
  Borrower email
</ParamField>

<ParamField body="Address_Street1" type="string">
  Street address
</ParamField>

<ParamField body="Address_City" type="string">
  City
</ParamField>

<ParamField body="Address_State" type="string">
  State
</ParamField>

<ParamField body="Address_Zipcode" type="string">
  Zip code
</ParamField>

<ParamField body="Loan_Amount" type="float">
  Loan amount
</ParamField>

<ParamField body="Rate" type="float">
  Interest rate
</ParamField>

<ParamField body="Origination" type="date">
  Origination date
</ParamField>

<ParamField body="Maturity" type="date">
  Maturity date
</ParamField>
