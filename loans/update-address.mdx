---
title: "Update a loan address"
description: "Updates an existing address on a loan"
api: "PATCH /loan/{id}/address/{address_id}"
---


Updates an existing address on a loan. Only include the fields you want to change. Any custom collateral fields defined in your product can also be updated by including them in the request body using their field name.

## Path Parameters

<ParamField path="id" type="string" required>
  The loan ID or loan number
</ParamField>

<ParamField path="address_id" type="string" required>
  The address ID to update
</ParamField>

## Request Body

<ParamField body="Address_Street1" type="string">
  Street address line 1
</ParamField>

<ParamField body="Address_Street2" type="string">
  Street address line 2
</ParamField>

<ParamField body="Address_City" type="string">
  City
</ParamField>

<ParamField body="Address_State" type="string">
  State
</ParamField>

<ParamField body="Address_Zipcode" type="string">
  Zip code
</ParamField>

<ParamField body="Address_Country" type="string">
  Country
</ParamField>

<ParamField body="Address_County" type="string">
  County
</ParamField>
