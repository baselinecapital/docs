---
title: "Add an address to a loan"
description: "Adds a new address to an existing loan"
---

`POST /loan/{id}/address`

Adds a new address to an existing loan. Address fields are set via body parameters. Any custom collateral fields defined in your product can also be set by including them in the request body using their field name.

## Path Parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `id` | string | Yes | The loan ID or loan number |

## Request Body

| Parameter | Type | Description |
| --- | --- | --- |
| `Address_Street1` | string | Street address line 1 |
| `Address_Street2` | string | Street address line 2 |
| `Address_City` | string | City |
| `Address_State` | string | State |
| `Address_Zipcode` | string | Zip code |
| `Address_Country` | string | Country |
| `Address_County` | string | County |

## Example Response

```json
{
  "loan": {
    "Id": "loan-123",
    "Name": "123 Main St",
    "Status": "servicing",
    "Addresses": [
      {
        "Id": "addr-456",
        "Street1": "123 Main St",
        "City": "New York",
        "State": "NY",
        "Zip": "10001",
        "Country": "United States",
        "County": "New York"
      }
    ]
  }
}
```
