---
title: "Add an address to a loan"
description: "Adds a new address to an existing loan"
api: "POST /loan/{id}/address"
---


Adds a new address to an existing loan. Address fields are set via body parameters. Any custom collateral fields defined in your product can also be set by including them in the request body using their field name.

## Path Parameters

<ParamField path="id" type="string" required>
  The loan ID or loan number
</ParamField>

## Request Body

<ParamField body="Address_Street1" type="string">
  Street address line 1
</ParamField>

<ParamField body="Address_Street2" type="string">
  Street address line 2
</ParamField>

<ParamField body="Address_City" type="string">
  City
</ParamField>

<ParamField body="Address_State" type="string">
  State
</ParamField>

<ParamField body="Address_Zipcode" type="string">
  Zip code
</ParamField>

<ParamField body="Address_Country" type="string">
  Country
</ParamField>

<ParamField body="Address_County" type="string">
  County
</ParamField>

## Example Response

```json
{
  "loan": {
    "Id": "loan-123",
    "Name": "123 Main St",
    "Status": "servicing",
    "Addresses": [
      {
        "Id": "addr-456",
        "Street1": "123 Main St",
        "City": "New York",
        "State": "NY",
        "Zip": "10001",
        "Country": "United States",
        "County": "New York"
      }
    ]
  }
}
```
