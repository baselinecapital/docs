`POST /loan`

All fields are optional. Any field available in the default product in your account can be set via a body parameter using its name. For example, a field named "Foo Bar" can be set by sending `{Foo_Bar:1234}` in the request.

## Request Body

| Parameter | Type | Description |
| --- | --- | --- |
| `Id` | string | (Optional) If not set, an ID will be generated automatically. |
| `Name` | string | Loan name |
| `Status` | enum | `lead`, `processing`, `underwriting`, `approved`, `closed`, `servicing`, `archived` (default: `servicing`) |
| `Borrower_Name` | string | Borrower full name |
| `Borrower_Email` | string | If a borrower with the same verified email already exists, that borrower will be attached. If not, a new borrower will be created. |
| `Borrower_Id` | string | If a borrower with the same ID already exists, that borrower will be attached. If not, a new borrower will be created. |
| `Address_Street1` | string | Street address |
| `Address_City` | string | City |
| `Address_State` | string | State |
| `Address_Zipcode` | string | Zip code |
| `Loan_Amount` | float | Loan amount |
| `Rate` | float | Interest rate |
| `Origination` | date (YYYY-MM-DD) | Origination date |
| `Maturity` | date (YYYY-MM-DD) | Maturity date |
| `Addresses` | array | Additional collateral addresses |
