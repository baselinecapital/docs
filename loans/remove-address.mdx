---
title: "Remove an address from a loan"
description: "Removes an address from a loan and deletes any associated collateral field values"
api: "DELETE /loan/{id}/address/{address_id}"
---


Removes an address from a loan and deletes any associated collateral field values. The address itself is unlinked from the loan.

## Path Parameters

<ParamField path="id" type="string" required>
  The loan ID or loan number
</ParamField>

<ParamField path="address_id" type="string" required>
  The address ID to remove
</ParamField>

## Example Response

```json
{
  "loan": {
    "Id": "loan-123",
    "Name": "123 Main St",
    "Status": "servicing",
    "Addresses": []
  }
}
```
