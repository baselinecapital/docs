---
title: "Update a loan document"
description: "Updates an existing document on a loan"
api: "PATCH /loan/{id}/document/{document_id}"
---


Updates an existing document on a loan. You can update both the document properties and its loan-specific properties (order, section, instructions).

<Warning>
  `Borrower_Id`, `Guarantor_Id`, and `Fund_Id` cannot be updated via this endpoint.
</Warning>

## Path Parameters

<ParamField path="id" type="string" required>
  The loan ID or loan number
</ParamField>

<ParamField path="document_id" type="string" required>
  The document ID to update
</ParamField>

## Request Body

<ParamField body="Name" type="string">
  Document name
</ParamField>

<ParamField body="Description" type="string">
  Document description
</ParamField>

<ParamField body="Status" type="string">
  Document status
</ParamField>

<ParamField body="Summary" type="string">
  Document summary
</ParamField>

<ParamField body="Is_Shared" type="boolean">
  Whether shared with borrower
</ParamField>

<ParamField body="Uploaded_Date" type="string">
  ISO 8601 upload date
</ParamField>

<ParamField body="Requested_Date" type="string">
  ISO 8601 requested date
</ParamField>

<ParamField body="Expired_Date" type="string">
  ISO 8601 expiry date
</ParamField>

<ParamField body="Shared_Date" type="string">
  ISO 8601 shared date
</ParamField>

<ParamField body="Order" type="integer">
  Display order
</ParamField>

<ParamField body="Section" type="string">
  Section the document belongs to
</ParamField>

<ParamField body="Instructions" type="string">
  Instructions
</ParamField>
